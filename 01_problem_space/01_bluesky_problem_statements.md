# BlueSky Problem Statements - Thriving Families

These broader framings are useful for inspiration. Teams should narrow them before building.

## Preparing Every Young Person for Their First Job — 17/27 (Needs work)
How might we use technology to ensure that every Richmond young person has access to the guidance, documents, and preparation they need to successfully pursue their first job — regardless of the strength of their adult support network?

Notes: Strong equity framing but broad. Preparation covers many dimensions (documents, labor law, transportation, interviews, financial literacy). Narrow to one dimension or one user type. D3 challenge: no compiled data package exists.

## Connecting Youth and Employers Across the City — 21/27 (Strong) ★ Recommended
How might we use technology to make it easier for Richmond youth to discover employment opportunities and for local employers to find prepared teen candidates — so that the match between willing workers and willing employers happens more reliably?

Notes: Clear two-sided problem with a software shape. Scope to discovery and matching guidance, not formal hiring. Employer confidence is a real friction point. Data challenge: no employer directory exists; must be curated.

## A Shared Understanding of Maternal Health — 15/27 (Needs work)
How might we use technology to help Richmond's maternal health stakeholders operate from a shared set of metrics and data sources so that duplicated analysis is reduced and advocacy is strengthened?

Notes: Important problem with clear organizational users. Lower score reflects unclear output type and unverified data source access. The four named sources (VDH, ACS, Kids Count, VDOE) need URL verification before building. Tool is for researchers, not patients.
